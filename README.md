## Deploy on Netlify

1. Create a new Netlify project and deploy this folder, or connect the folder through a Git repository.
2. In Netlify, open **Forms** and enable form detection.
3. Redeploy the site so Netlify detects the form named `quote`.
4. Add a form submission email notification for `info@axiosconstruction.com.au`.
5. Add `axiosconstruction.com.au` and `www.axiosconstruction.com.au` under Domain management, then follow the displayed DNS instructions.
6. Submit a test enquiry and confirm receipt before advertising the site.
